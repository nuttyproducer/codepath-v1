Accessibility — Keyboard Navigation

Skip links, :focus-visible system, tab order, and accessible accordion.

Exercise 1 — Skip link

Press Tab while this page is loaded (no pre-focus). The first Tab keypress should reveal the skip link in the top-left corner. Pressing Enter should jump directly to the main content, skipping the navigation entirely.

How to test: Click in the browser address bar, then press Tab. You should see the skip link appear. Press Enter to activate it. Press Tab again — focus should be inside #main-content.

Exercise 2 — :focus-visible ring system

Tab through the interactive elements below. Notice that only keyboard focus shows a ring — mouse clicks do not trigger the visible outline.

:focus fires for ALL focus (mouse, touch, keyboard). :focus-visible fires only when the browser determines a visible focus indicator is needed — typically keyboard navigation.

Exercise 4 — Accessible accordion with <details>

The native <details> element is keyboard-accessible out of the box. It can be focused, opened with Enter/Space, and screen readers announce its state. Style it to look like a custom accordion.

What is HackYourFuture Belgium? ▶

HackYourFuture Belgium is a free coding school for refugees and newcomers. We teach web development skills to help people build new careers.

Who can apply? ▶

Applications are open to refugees, asylum seekers, and other newcomers who face barriers to employment. Basic computer literacy is required.

What will I learn? ▶

The curriculum includes HTML, CSS, JavaScript, React, Node.js, and databases. The programme runs over 8 months and includes real project work.

Is there a cost? ▶

No — the programme is entirely free for students. HackYourFuture Belgium is funded by grants, donations, and partnerships with companies.

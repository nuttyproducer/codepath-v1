Exercise 2 — Multi-column text with column-count

HackYourFuture Belgium was founded in 2017 to provide coding education to refugees and newcomers. The curriculum covers the full web development stack, from HTML and CSS foundations to JavaScript, Node.js, and databases.

Students commit to a rigorous 8-month programme, attending classes on weekends while often working or studying during the week. The assignments in this collection are designed to be completed progressively, each building on skills from the previous exercise.

CSS multi-column layout is ideal for magazine-style articles where you want text to flow naturally across columns. The column-rule property adds a visual divider between columns, similar to a border, but without affecting layout flow.

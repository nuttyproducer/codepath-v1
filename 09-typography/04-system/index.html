Exercise 3 — Vertical rhythm

Vertical rhythm is the consistent spacing between text elements, creating a harmonious flow down the page. Use a base spacing unit (e.g. --space: 1rem) and derive all spacing from it.

Module overview

Students begin with HTML fundamentals, learning semantic markup and document structure. This foundation is critical for all following modules.

CSS foundations

The cascade, specificity, and the box model. Students progress from inline styles to external stylesheets, and from simple selectors to pseudo-classes.

Responsive design

Mobile-first methodology, fluid layouts with Flexbox and Grid, and media queries. Students build layouts that work on any screen size.
